## Дальше

1. Получить две фотографии в доступной рабочей папке или как вложения чата.
2. Добавить их в `assets/projects/` под понятными именами, например `renovation-lihachevsky.jpg` и `renovation-raduzhnaya.jpg`.
3. Обновить пути в объекте `projectImages` в `Проекты.dc.html` при необходимости.
4. Открыть страницу проектов и проверить обе карточки и оба модальных окна на desktop и mobile.
5. Проверить опубликованную версию GitHub Pages после обновления.

## Точка продолжения

Откройте `Проекты.dc.html` и найдите объект `projectImages` в нижней части файла. Сейчас используются:

```js
'renovation-lihachevsky': './assets/projects/renovation-lihachevsky.jpg',
'renovation-raduzhnaya': './assets/projects/renovation-raduzhnaya.png',
```

После получения файлов загрузите их в `assets/projects/`. Для Лихачёвского переулка можно заменить существующий файл с тем же именем; для Радужной улицы либо заменить существующий PNG, либо использовать новый JPG и обновить путь. Карточки и модальные окна берут изображение из этого единственного объекта, поэтому отдельная правка HTML-разметки не нужна. Затем убедитесь, что фото не обрезается критично при `object-fit: cover`.
